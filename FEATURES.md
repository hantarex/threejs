# CS:GO Weapon Skin Viewer — Функциональность

## ✨ Основные возможности

### 3D Визуализация
- **Процедурная модель AK-47** — детальная геометрия с 12+ компонентами
- **Процедурные текстуры** — металл с царапинами, деревянный stock
- **HDR окружение** — реалистичное освещение от Polyhaven
- **Bloom постобработка** — сияние на металлических деталях
- **ACES тоновое отображение** — кинематографический вид

### Интерактивность
- **Свободное вращение** — drag мышью в любом направлении
- **Zoom** — колёсико мыши для приближения/удаления
- **Автоматическое вращение** — медленное вращение по умолчанию
- **Damping эффект** — гладкая инерция при отпускании

### Режимы просмотра
- **Free View** — полный обзор модели с расстояния (2.5 units)
  - Большой диапазон zoom (0.8 - 6.0)
  - Медленное автоматическое вращение (1.2 speed)
  
- **Inspect** — близкий осмотр деталей (1.2 units)
  - Узкий диапазон zoom (0.3 - 2.0)
  - Ускоренное вращение (0.8 speed)
  - Идеален для рассмотрения текстур и деталей

### UI/UX
- **Красивый overlay** — название оружия и скина в CS:GO стиле
- **Кнопки переключения режимов** — интерактивные, с feedback
- **Loading bar** — прогресс загрузки модели
- **Подсказки** — как управлять моделью
- **Responsive дизайн** — адаптивно под любой размер экрана

## 🎨 Визуальный стиль

### Цветовая схема
- Фон: `#0a0a0f` (глубокий чёрный)
- Названия: `#f0f0f0` (белый)
- Скин: `#e8c66a` (CS:GO золото)
- Кнопки активные: золотой фон с чёрным текстом
- Кнопки неактивные: полупрозрачный белый фон

### Материалы
- **Металл** (receiver, barrel) — metalness 0.92, roughness 0.25
- **Дерево** (stock) — metalness 0.05, roughness 0.85
- **Сталь** (barrel) — metalness 0.98, roughness 0.15
- **Текстуры** — процедурно сгенерированные canvas текстуры

## 🔧 Техническая реализация

### Архитектура
```
main.js           → Bootstrap & инициализация
  ↓
scene.js          → Three.js сцена, камера, renderer
lighting.js       → HDR + directional lights
loader.js         → GLTFLoader для моделей
controls.js       → OrbitControls
postprocessing.js → EffectComposer + Bloom
fallback.js       → Процедурная модель AK-47
ui.js             → UI overlay + кнопки
```

### Производительность
- **JS бандл**: 636KB (Three.js included)
- **CSS**: 1.55KB
- **HDR карта**: 1.5MB
- **GLB модель**: процедурная (0 KB внешних файлов)
- **Render**: 60 FPS на современных GPU

### Оптимизации
- Pixel ratio capped на 2x (для high-DPI экранов)
- PCFShadow mapping для мягких теней
- ACES tone mapping вместо Linear
- Bloom с threshold 0.85 (только яркие блики)

## 📱 Мобильная совместимость
- Полная поддержка touch событий
- OrbitControls работает с двумя пальцами
- Responsive UI без overflow
- Работает на планшетах и мобильных

## 🚀 Установка готовой GLB модели

Если хочешь заменить процедурную модель на готовую GLB:

1. Скачай GLB модель:
   - [Poly Pizza](https://poly.pizza/search/rifle)
   - [Sketchfab](https://sketchfab.com/search?q=ak47)
   - Любой другой источник Three.js совместимых моделей

2. Переименуй в `scene.glb` и положи в:
   ```
   public/models/ak47/scene.glb
   ```

3. Перезагрузи страницу!

Код автоматически:
- Загрузит новую модель
- Отцентрирует её в сцене
- Нормализует масштаб
- Применит освещение и материалы
